# Cidadania Digital e Inteligência Artificial: O Impacto das Deepfakes e da Desinformação na Sociedade

Este projeto consiste em um portal web interativo e funcional focado em conscientizar a comunidade escolar e a sociedade sobre os perigos trazidos pela desinformação gerada por IA (Deepfakes), instruindo os cidadãos a identificarem mídias manipuladas.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo (`<main>`, `<section>`, `<footer`, etc).
* **CSS3:** Estilização baseada em variáveis nativas, layout moderno com Flexbox e responsividade via Media Queries.
* **JavaScript (Vanilla):** Manipulação dinâmica do DOM para o seletor de acessibilidade (Modo Escuro) e validação lógica de respostas do quiz informativo.

## 🛠️ Comandos e Prompts de IA Utilizados
Durante o desenvolvimento das etapas deste projeto, os seguintes prompts estruturados foram utilizados para geração de ideias e refinamento técnico:

1. **Para Design de Arquitetura de Software:** *"Gere uma estrutura semântica de HTML5 para um portal focado em cidadania digital e riscos de deepfakes, contendo seção hero, cards informativos e um formulário de quiz."*
2. **Para Lógica JavaScript:** *"Escreva um script JavaScript limpo que capture a submissão de um formulário de quiz, valide a alternativa correta e exiba mensagens estilizadas de sucesso ou erro na tela sem recarregar a página."*
